## 三、代码架构基线

### 3.1 源码规模

```
iptv-edge-agent/app/src/main/java/com/joctv/agent/
├── MainActivity.kt                 ~2170 行（主编排中心）
├── asr/                            12 文件（ASR 识别链路）
├── tts/                            14 文件（TTS 合成链路）
├── audio/                          8 文件（音频前端处理）
├── digitalhuman/                   9 文件（数字人渲染）
│   └── taoavatar/                  3D 数字人（当前主用）
├── conversation/                   3 文件（对话状态机）
├── intent/                         1 文件（意图路由）
├── engine/                         4 文件（早期抽象，部分未使用）
├── realtime/                       6 文件（实时数据查询）
├── web/                            1 文件（LLM SSE 客户端）
└── utils/                          2 文件（工具类）
```

### 3.2 核心数据流（已对齐）

```
AudioRecord(16kHz mono)
  → WakeGateController (IDLE: 轻量VAD / CAPTURE: 完整ASR)
  → SimpleGate (参考信号回声门控)
  → VadDetector (能量VAD, 800ms静音)
  → NpuOfflineAsrEngine (SenseVoice RKNN, NPU)
  → IntentRouter (实时查询 > 本地命令 > LLM)
  → TtsOrchestrator (分句 → 双引擎合成)
  → TaoAvatarController (PCM → A2BS → NNR渲染)
```

### 3.3 对话状态机

```
IDLE →[唤醒词]→ LISTENING →[ASR结果]→ READY →[收到回复]→ SPEAKING
↑                                                        │
└──[8秒超时]── COOLDOWN ←──[TTS完成]────────────────────┘
                              ↑ │
                              │ └──[8秒内继续说]→ READY → SPEAKING (无需唤醒)
```
